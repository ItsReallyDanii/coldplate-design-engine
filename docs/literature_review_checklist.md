# Literature Review Checklist

**Stage: 0.5 — in progress.**

Record citations in `claim_audit_v2/` for each completed item. Use dry, traceable sources only.

- [ ] Cold plate performance metrics (junction-to-coolant thermal resistance, coefficient of performance, temperature uniformity definitions).
- [ ] Direct-to-chip liquid-cooling constraints (footprint, interface layout, allowable pressure budget, heat-load envelopes).
- [ ] Coolant assumptions (fluid selection, property ranges, chemistry compatibility with cold-plate materials).
- [ ] Thermal resistance reporting conventions (reference locations, normalization methods, uncertainty treatment).
- [ ] Pressure drop reporting conventions (reference planes, measurement locations, normalization methods).
- [ ] TPMS cold-plate literature (gyroid, diamond, primitive implementations and measured/simulated performance).
- [ ] Channel baseline literature (straight parallel, serpentine, pin-fin / interrupted channel data and correlations).
- [ ] Functionally graded heat exchangers (published designs, performance claims, and manufacturing methods).
- [ ] Spatially varying TPMS (literature on graded unit-cell size, graded porosity, or continuous-gradient TPMS lattices).
- [ ] Multi-zone / segmented cold plates (cold plates with discrete thermal zones or segmented lattice regions designed for non-uniform loads).
- [ ] Additive manufacturing constraints for graded porosity (AM process limits for continuous porosity variation, minimum feature size as a function of local porosity).
- [ ] Powder evacuation / trapped volume constraints (AM design rules for internal voids, support removal, powder removal access in graded or complex internal structures).
- [ ] Non-uniform hotspot maps in GPU cold plates (published thermal load distributions, hotspot locations, and thermal design strategies for GPU or accelerator cold plates).
- [ ] Manufacturability constraints (CNC and AM feature limits, post-processing requirements, surface finish effects).
- [ ] Test-rig conventions (instrumentation, boundary-condition control, calibration practices).
- [ ] CFD validation practices (mesh independence, turbulence/porous-model selection, solver settings).
- [ ] FEA validation practices (load mapping from CFD, boundary conditions, material property sources).
