# Reading Queue

Status values (progression: `not found` → `queued` → `reading` → `captured` → `summarized` → `incorporated`):
- `not found`: target source not yet located.
- `queued`: located and awaiting read.
- `reading`: actively being read.
- `captured`: source data entered in the capture template.
- `summarized`: capture template completed with summary.
- `incorporated`: insights applied to design choices or claims.
